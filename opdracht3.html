3. Array van objecten

3.1 Array definieren

Definieer de constante docenten,
met als waarde een array met 3 objecten.
Elk van deze objecten stelt een docent voor en heeft de volgende eigenschappen:

  • voornaam
  • achternaam
  • hobbies (dit is een array met tenminste 2 hobbies)

3.2. Array doorlopen

Toon hieronder de namen (voornaam en achternaam) van de docenten (gebruikmakend van de Array),
zet een <br> na elke naam.

3.3. Random docent

Toon bij een druk op de onderstaande knop, een random naam van de docenten.

3.4. Random hobby van random docent

Toon bij een druk op de onderstaande knop, een random hobby van een random docent.
